📌 Difficulty: Easy  
📚 Topic: Array, Two Pointers  
🖥️ Language: Java

## 💡 Approach
- Used a two-pointer technique to overwrite elements equal to the target.
- Maintained a pointer for the position of non-target elements.
- Operated in-place to reduce memory usage.
