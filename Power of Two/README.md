# 2 - Power of Two

🔗 [LeetCode Problem Link](https://leetcode.com/problems/power-of-two/)  
📌 Difficulty: Easy  
📚 Topic: Bit Manipulation  
🖥️ Language: Java

## 💡 Approach
- Checked if `n` is positive and whether `n & (n - 1)` equals 0.
- This works because power of two numbers have only one bit set.
